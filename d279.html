D - Compressed Files:

 

If you are using HTML and CSS with images, please make sure to compress the files into a zip

 

To compress a file into a zip file in Windows 10 or 11, you can do the following: 

 

  • Find the file or folder you want to compress 
  • Right-click or press and hold the file or folder 
  • Select Send to 
  • Select Compressed (zipped) folder 

 To compress files into a zip on a Mac, you can do the following: 

 

  • Locate the file or folder you want to compress 
  • Right-click on the file or folder 
  • Select Compress from the shortcut menu
